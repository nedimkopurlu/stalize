## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Arrow-to-number gap within a price change inline span |
| sm | 8px | Gap between forex pair rows; gap between gold item rows; label-to-value inside a widget cell |
| md | 16px | Default padding inside widget cells; gap between BIST100 banner metrics |
| lg | 24px | Card inner padding (matches `.card` class: `padding: 24px`); gap between major sections |
| xl | 32px | Vertical margin below the BIST100 banner before the forex/gold grid |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

Exceptions:
- BIST100 banner: inner padding 24px all sides, but the change value display uses 0 inline gap (number and arrow are a single inline element).
- Forex/Gold grid column gap: 16px desktop; collapses to 0 (single column) on mobile — rows stack with 8px row gap.
- Portföy placeholder card: inner padding 24px, dashed border style to signal "future" state.

Source: globals.css `.card` (24px), `.gap-*` utility classes, CONTEXT.md D-04/D-05

## Typography

| Role | Size | Weight | Line Height | Font | Usage |
|------|------|--------|-------------|------|-------|
| Display | 28px (1.75rem) | 700 | 1.2 | monospace (`var(--font-mono)`) | BIST100 endeks değeri |
| Heading | 20px (1.25rem) | 700 | 1.3 | body stack | Section headers ("Döviz", "Altın", "Portföy") |
| Body | 16px (1rem) | 400 | 1.6 | body stack | Widget labels, placeholder text |
| Label / Mono-value / Change / Meta | 14px (0.875rem) | 700 | 1.0 | body stack or monospace (see Usage) | Döviz çifti adı, altın tipi adı, tüm sayısal fiyat değerleri, % değişim değerleri; Meta labels additionally use `text-transform: uppercase; letter-spacing: 0.05em` (hacim etiketi, widget eyebrow labels) |

Rules:
- Tüm sayısal değerler (fiyat, hacim, yüzde) `font-family: var(--font-mono), monospace` kullanır.
- BIST100 günlük değişim yüzdesi: 28px display size, weight 700, yeşil/kırmızı renk.
- Hacim etiketi: 14px, weight 700, `--text-muted` rengi, uppercase, letter-spacing 0.05em.
- Portföy placeholder metni: 14px body weight 400, `--text-secondary`.
- Font weight scale: exactly 2 weights — 400 (body/placeholder text) and 700 (all emphasis, headings, labels, values, display).

Source: globals.css `.statValue` (1.75rem), `.signalCard strong`, `.heroMetric strong`; adapted down for widget cells.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `--bg-primary: #0a0f1e` | Sayfa arka planı |
| Secondary (30%) | `--bg-card: rgba(16, 28, 45, 0.82)` | Tüm widget kartları (BIST100 banner, döviz kartı, altın kartı, portföy kartı) |
| Accent (10%) | `--accent-cyan: #7dd3fc` | Sadece: sayfa başlığı gradient'ı, BIST100 banner eyebrow etiketi, yenileme timer ikonu/text |
| Positive | `--green-400: #4ade80` | ▲ sembolü ve % değişim sayısı: pozitif değerler |
| Negative | `--red-400: #f87171` | ▼ sembolü ve % değişim sayısı: negatif değerler |
| Destructive | N/A — Bu fazda yıkıcı aksiyon yoktur | — |

Accent reserved for: sayfa başlığı gradyan metni, BIST100 banner küçük eyebrow etiketi ("BIST100 ENDEKSİ"), 30 saniyelik otomatik yenileme countdown/status göstergesi. Döviz ve altın fiyat değişimlerinde kesinlikle kullanılmaz — bu alanlar yalnızca `--green-400` / `--red-400` ile ifade edilir.

Nötr metin:
- Birincil içerik: `--text-primary: rgba(255, 255, 255, 0.95)`
- İkincil içerik (etiketler): `--text-secondary: rgba(255, 255, 255, 0.60)`
- Soluk meta: `--text-muted: #8ca0b8`

Arka plan glassmorphism: `backdrop-filter: blur(20px)` tüm `.card` elemanlarında (globals.css tanımlı, dokunulmaz).

Portföy placeholder kart sınırı: `border: 1px dashed rgba(148, 163, 184, 0.20)` — solid değil dashed, "henüz dolu değil" sinyali.

Source: globals.css CSS custom properties, CONTEXT.md D-06/D-07/D-08/D-12

## Layout Contract

### Bölüm Sırası (yukarıdan aşağıya)

```
AppShell (Sidebar + main wrapper)
  └── TerminalPageHeader (title + description)
  └── BIST100 Banner (full-width)
  └── İkili Grid: [Döviz Widgetı] [Altın Widgetı]  ← 2 kolon masaüstü, 1 kolon mobil
  └── Portföy Özet Widgetı (full-width)
```

Source: CONTEXT.md D-03/D-04/D-05

### BIST100 Banner

- Full-width `.card` sınıfı
- İçerik: sol taraf `XU100` etiketi + endeks değeri (display 28px mono), sağ taraf: % değişim (büyük, renkli), orta/sağ: Hacim etiketi + formatVolume() değeri
- Düzen: `display: flex; justify-content: space-between; align-items: center; flex-wrap: wrap; gap: 16px`
- Mobilde: flex-direction column olmaz; wrap ile küçülür, 3 değer tek satırda durmazsa 2+1 olarak kırılır

### Forex / Gold Grid

- CSS Grid: `grid-template-columns: 1fr 1fr; gap: 16px`
- Breakpoint ≤768px: `grid-template-columns: 1fr`
- Her widget `.card` sınıfı
- İç liste: her satır `display: flex; justify-content: space-between; align-items: center; padding: 8px 0; border-bottom: 1px solid var(--border-glass)`; son satırda border-bottom yok

### Forex Widget Sırası

USD/TRY → EUR/TRY → GBP/TRY → CNY/TRY → JPY/TRY → CHF/TRY

Türkçe etiketler: "Dolar", "Avro", "Sterlin", "Çin Yuanı", "Japon Yeni", "İsviçre Frangı"

### Altın Widget Sırası

Gram Altın → Ons Altın → Çeyrek Altın → Yarım Altın → Tam Altın

Türkçe etiketler: "Gram Altın", "Ons Altın", "Çeyrek Altın", "Yarım Altın", "Tam Altın"

Fiyat birimi: "₺" prefix, formatPrice() ile 2 ondalık basamak (tr-TR locale, binlik nokta ayracı).

## Interaction Contract

### Otomatik Yenileme

- İlk yükleme: `useEffect` ile sayfa mount'ta `Promise.allSettled([bist100, forex, gold])` çağrısı
- Her endpoint bağımsız state: `bist100State`, `forexState`, `goldState` — biri başarısız diğerleri etkilenmez
- `setInterval(refresh, 30_000)` + `return () => clearInterval(id)` cleanup
- Görsel gösterge: widget header sağında küçük "son güncelleme: HH:MM:SS" veya "30s" countdown text — `--accent-cyan` rengi, 14px mono, uppercase, letter-spacing 0.05em
- Yenileme sırasında widgetlar DOM'dan kaldırılmaz; mevcut veri yerinde kalır (no flash/unmount)

### Loading State

- İlk yüklemede (henüz veri gelmemiş): her widget alanı için skeleton placeholder
- Skeleton: `className="skeleton"` global class kullanılır (globals.css `@keyframes shimmer` ile tanımlı)
- BIST100 skeleton: tam genişlikte, yükseklik 80px
- Forex skeleton: 6 satır, her biri 20px yükseklik, 8px gap
- Gold skeleton: 5 satır, aynı pattern
- Yenileme sırasında (veri zaten var): skeleton gösterilmez — sessiz arka plan güncelleme

### Error State

- Her widget bağımsız hata state'i: `forexError: string | null`
- Hata durumunda widget kartının içinde `TerminalError` primitive kullanılır
- Error mesajı: "Döviz verisi alınamadı. Sayfa yenilendiğinde tekrar denenecek."
- Hata kartı sınırı yok (`--red-400` renk kullanılmaz widget sınırında) — yalnızca içerideki TerminalError bileşeni renkli

## Sayı Formatı

- Fiyat: `formatPrice(value)` — tr-TR locale, 2 ondalık, binlik nokta (örn: "32.450,75")
- Yüzde değişim: `formatPercentage(value)` ile format, ardından `PriceChange` bileşeni ▲/▼ arrow prefix'i ekler
- Hacim: `formatVolume(value)` — kısaltma formatı (örn: "1,2M", "450K")
- Null/eksik değer: "—" (em-dash, tek karakter)

Source: StockHelpers.tsx `formatPrice`, `formatVolume`, `formatPercentage`, `PriceChange`
