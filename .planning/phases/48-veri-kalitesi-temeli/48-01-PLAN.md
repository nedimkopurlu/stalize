    1) Add the function to `backend/app/services/scoring.py` (top-level module function, NOT inside ScoringEngine, so it can be imported standalone):

       ```python
       def calculate_data_quality_score(fundamental) -> float:
           """0-100 data quality score for a Fundamental row.

           Starts at 100. For each present-but-USD-suspicious field, subtract 30.
           For each null fundamental field, subtract 10. Clamped to [0, 100].

           USD-suspicious thresholds (BIST values converted to USD look unrealistically small):
             - pe_ratio: 0 < pe < 2 OR pe < 0.5 (i.e., pe is not None and pe < 2)
             - pb_ratio: pb is not None and pb < 0.05
           Null-penalty fields: pe_ratio, pb_ratio, ev_ebitda, net_profit, revenue
           """
           if fundamental is None:
               return 0.0
           score = 100.0
           pe = getattr(fundamental, "pe_ratio", None)
           pb = getattr(fundamental, "pb_ratio", None)
           ev_eb = getattr(fundamental, "ev_ebitda", None)
           net_profit = getattr(fundamental, "net_profit", None)
           revenue = getattr(fundamental, "revenue", None)

           # USD-suspicious detection (only when value present)
           if pe is not None and pe < 2:
               score -= 30
           if pb is not None and pb < 0.05:
               score -= 30

           # Null penalty
           for field_val in (pe, pb, ev_eb, net_profit, revenue):
               if field_val is None:
                   score -= 10

           return max(0.0, score)
       ```
