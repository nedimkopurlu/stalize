- BREF-05: Net AI commentary — 3-4 sentence risk/opportunity summary for the day. Generated by DeepSeek via instructor (`DailyCommentary` Pydantic model with `risk_summary`, `opportunities`, `watch_list` fields).

<specifics>
## Specific Ideas

- Use `ON CONFLICT (date) DO UPDATE SET ...` (PostgreSQL UPSERT) for idempotent job reruns
- `GET /api/briefing/today` returns 404 with `{"detail": "Brifing henüz üretilmedi"}` if no record exists for today
- DailyCommentary generated with `instructor.from_openai(...)` but as a direct call (not via `analyze()`) — briefing_generator creates its own instructor client or calls a new method on DeepSeekSentimentService
- Notable stocks threshold: volume > 2× 20-day average OR abs(daily_change_pct) > 3.0
- Time zone for cron: APScheduler timezone should be "Europe/Istanbul" (UTC+3) for 06:30 local time
- Briefing endpoint: include `X-Cache: HIT` header when served from DB (always — no LLM at request time)
