```python
def _compute_ema_trend_score(self, df: pd.DataFrame) -> float:
    """Returns 0-50 float EMA trend component.
    
    Scoring rules:
    - price > ema_200:   +20 base points
    - ema_50 > ema_200:  +15 additional (golden cross zone)
    - momentum bonus:    ((close - ema_200) / ema_200).clip(0, 0.10) * 150 → 0-15 points
    - total range:       0-50 points
    - Returns 0.0 if ema_200 or ema_50 is NaN (insufficient data)
    """
```

Blend formula in calculate_score():
- signal_score: existing 0-100 from bullish/bearish signal balance
- ema_component: 0-50 from _compute_ema_trend_score(), normalized to 0-100 by * 2
- blended: signal_score * 0.6 + ema_score_normalized * 0.4
- Result must stay in [0, 100]
- This keeps the overall technical_score in 0-100 without changing scoring.py weights
