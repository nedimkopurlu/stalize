**Endpoint 1 — GET /api/portfolio/positions (MPRT-03):**
```
- is_active=True olan tüm pozisyonları çek
- Her pozisyon için yfinance'den güncel fiyat çek (symbol + ".IS")
- pnl_pct = (current_price - entry_price) / entry_price * 100
- Dönen alan başına: id, symbol, entry_price, quantity, entry_date,
  stop_loss, target_price, rationale, current_price, pnl_pct
- Fiyat çekilemezse current_price=None, pnl_pct=None döner (hata fırlatma)
```

**Endpoint 2 — GET /api/portfolio/history (MPRT-03):**
```
- Query param: days: int = Query(90, ge=1, le=365)
- Son N günün portfolio_daily_snapshots kayıtlarını döner
- date DESC sıralı
- Dönen: id, date, total_value, daily_pnl_pct, positions_json (ham JSONB)
```

**Endpoint 3 — POST /api/portfolio/positions (MPRT-05):**
```
- PositionCreate body alır
- symbol büyük harfe çevir (.upper())
- is_active=True olarak portfolio_positions tablosuna yeni kayıt ekle
- portfolio_change_log'a ADD kaydı yaz:
    date=bugün, action="ADD", symbol=symbol, reason=rationale
- 201 status kodu ile oluşturulan kaydın id'sini döner
```

**Endpoint 4 — DELETE /api/portfolio/positions/{position_id} (MPRT-05):**
```
- position_id ile PortfolioPosition'ı çek — bulunamazsa 404
- is_active=False yap (soft delete — kayıt silinmez)
- portfolio_change_log'a REMOVE kaydı yaz:
    date=bugün, action="REMOVE", symbol=pos.symbol, reason=None
- 200 status ile {"status": "closed", "symbol": pos.symbol} döner
```

**Endpoint 5 — POST /api/portfolio/change-log (MPRT-05):**
```
- ChangeLogCreate body alır
- action değeri "ADD" veya "REMOVE" dışındaysa 422 döner
- Doğrudan portfolio_change_log tablosuna kayıt ekle
- 201 status ile oluşturulan kaydın id'sini döner
```

Tüm endpoint'lerde:
- `async with AsyncSessionLocal() as db:` pattern'ini kullan
- Exception'ları yakala, 500 yerine anlamlı HTTP hata kodları döner
- logger.info ile işlem logla
