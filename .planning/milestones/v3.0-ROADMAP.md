# Milestone v3.0: Tüm Borsa + Kullanılabilir Platform

**Status:** ✅ SHIPPED 2026-04-28
**Phases:** 17–21
**Total Plans:** 9

## Overview

v2.0, gerçek veri omurgasını kurdu. v3.0 bunu gerçekten kullanılabilir bir yatırım araştırma ve takip platformuna dönüştürdü:
- BIST100'den (~100) tüm Borsa İstanbul'a (~400 aktif hisse) genişleme
- Her hisse sayfasında TradingView grafik + rakip karşılaştırma + EV/EBITDA
- Çok boyutlu tarama motoru (screener) — 4 hazır şablon + özelleştirilebilir builder
- İzleme listesi (watchlist) — localStorage tabanlı, hisse sayfasından toggle
- Sistem teması auto-detect (prefers-color-scheme), sidebar güncelleme

## Milestone Summary

**Key Decisions:**
- Watchlist localStorage tabanlı — backend gerektirmiyor, kişisel kullanım için yeterli
- bist_full_universe.py statik liste — yılda 1-2 güncelleme yeterli, manuel kontrol kolay
- TradingView iframe embed — free widget, BIST:SYMBOL format çalışıyor
- Screener iki aşamalı — SQL filtre (Stock) + Python post-filtre (Fundamental), daha temiz
- bist100_universe.py korundu — testler bağımlı (104 hisse), dokunulmadı

**Technical Debt:**
- Phase 19 glassmorphism CSS kaldırma deferred (kullanıcı mevcut tasarımı beğeniyor)
- Watchlist backend persistence yok (localStorage only) — v4.0 için düşünülecek
- Screener performance optimization (500 hisse için sub-query join) — v4.0

**Issues Resolved:**
- BIST100 kısıtı kaldırıldı → tüm Borsa İstanbul
- yfinance rate limiting ile 400+ hisse güvenli çekilebilir hale getirildi
- Hisse listesi 100'den fazla hisse ile kullanılabilir (infinite scroll)
