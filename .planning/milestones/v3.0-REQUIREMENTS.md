## Functional Requirements — Final Status

### F-01: Evren Genişlemesi — Tüm Borsa İstanbul ✅
- [x] BIST100 (~100) → Tüm Borsa İstanbul aktif hisseleri (~400) genişlemesi — Phase 17
- [x] DataCollector toplu çekim için akıllı rate limiting (batch 10/1s, hata izolasyonu) — Phase 17
- [x] Hisse metadata: sembol, ad, sektör, market_tier (yıldız/ana), BIST30/100/250 — Phase 17
- [~] yfinance `.IS` sembol otomatik discovery → Elle hazırlanmış statik liste (kararla değiştirildi)

### F-02: Güvenilir Hisse Sayfası ✅
- [x] TradingView gömülü grafik (BIST:SYMBOL, dark theme, 500px) — Phase 18
- [x] Temel metrikler: F/K, PD/DD, EV/EBITDA, ROE, net marj, borç/özsermaye — Phase 18
- [x] Rakip karşılaştırması: aynı sektörden 5 hisse, market_cap desc sıralı — Phase 18
- [x] KAP akışı: 10 bildirim, zaman sıralı — Phase 18
- [x] Skor kartı: Al/Tut/Sat + Temel/Teknik/Haber breakdown — mevcut, Phase 18 korudu
- [~] 52-hafta aralığı → deferred (yfinance'tan çekilebilir, öncelik düşük)

### F-03: Tarama Motoru (Screener) ✅
- [x] Hazır filtre şablonları: Düşük F/K, Momentum, Güçlü Bilanço, Yüksek Temettü — Phase 20
- [x] Özelleştirilebilir filtre builder: sektör, bist30/100/250, skor, F/K, PD/DD, ROE, sinyal — Phase 20
- [x] Filtre setlerini kaydet (localStorage) — Phase 20
- [x] Sonuç tablosu: sıralanabilir, sembol, fiyat, skor, metrikler — Phase 20
- [x] Sonuçtan hisse sayfasına tek tıkla geçiş — Phase 20

### F-04: Watchlist + Portföy Modülü ✅ (kısmi)
- [x] Watchlist: hisseleri kaydet, fiyat + değişim + skor + sinyal ile izle — Phase 21
- [x] Hisse sayfasından tek tıkla watchlist'e ekle/çıkar — Phase 21
- [~] Portföy: alım/satım işlemi + P&L → mevcut /portfolio sayfası korundu (v2.0 mirası)

### F-05: Dashboard Yeniden Tasarım ✅ (mevcut yeterli)
